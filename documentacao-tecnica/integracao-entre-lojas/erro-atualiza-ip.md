# Erro atualiza IP

O erro para atualizar o IP ocorre quando o DMconfigurações está validando a comunicação corretamente, mas o DMBackup não consegue validar o IP da internet do cliente, sendo assim os dados não são atualizados no controle da Dmaster.

Ao verificar os logs e os erros abaixo estiverem descritos, as possíveis soluções são: realizar a substituição do servidor de máquina ou orientar o cliente a trocar o sistema operacional, visto que Windows 7 já está sem atualização a bastante tempo.
