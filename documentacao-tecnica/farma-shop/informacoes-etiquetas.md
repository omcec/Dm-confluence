# Informações etiquetas

Através do sistema DMASTER FARMA/SHOP, é possível realizar a impressão de alguns tipos de etiqueta. Estes serão explicados a seguir, juntamente com as informações que são exibidas em cada uma delas.

### Etiqueta produto

Tipo de etiqueta normalmente utilizada para inserir preço nos produtos individualmente, ou seja, é impressa uma etiqueta para cada unidade do produto.

As informações impressas nesse tipo de etiqueta são:

* Nome do estabelecimento
* Telefone
* Código de barras, descrição e preço do produto

Para impressão desse tipo de etiqueta, é necessário inserir o número da linha e coluna onde a etiqueta está, de acordo com a folha de etiquetas que o cliente possui. Segue exemplo:

![](https://981618378-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FaHl55K4HG6CuWy9zUjR9%2Fuploads%2FsL1Fpopp87b1vbXSJDvS%2Fetq%20linha%20e%20coluna.png?alt=media\&token=2d437917-330a-41c1-9dd8-06288ed56f89) ![](https://981618378-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FaHl55K4HG6CuWy9zUjR9%2Fuploads%2FnPNaUVGq004MQQn4qayx%2Ftempsnip.png?alt=media\&token=bd54992a-6ea5-4a77-8dec-1d70fda13920)

A etiqueta pode ser impressa com ou sem o preço do produto. Isso é definido na hora da impressão, dependendo de qual opção o usuário selecionar no módulo de etiquetas **(Etiq. Produto Sem Preço ou Etiq. Prod Com Preço)**.

![](https://981618378-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FaHl55K4HG6CuWy9zUjR9%2Fuploads%2F3x1vwsqogfH2i1IQosU2%2Ftipo%20etq%20produto.png?alt=media\&token=48db531c-9424-4c2e-b6fe-39594625bcd7)

Nesta etiqueta, o correto é imprimir o código de barras do produto.\
Caso a etiqueta esteja sendo impressa com o código interno do produto, é necessário instalar a fonte **free3of9.ttf**, que fica na pasta **MBHSist.**

{% hint style="info" %}
Para instalar a fonte **free3of9.ttf**, é necessário clicar duas vezes sobre a mesma e na tela que irá abrir clicar em **instalar**. Após esse processo, reiniciar o sistema.
{% endhint %}

* Para realizar impressão desse tipo de etiqueta é utilizada **impressora Laser**. **Impressoras multifuncionais** também podem ser utilizadas, mas neste caso, a qualidade da impressão pode não ser tão boa.
* A impressão desse tipo de etiqueta é realizada na **impressora configurada como padrão**, ou seja, a impressora laser instalada no computador deve estar configurada como padrão no menu dispositivos e impressoras do windows.
{% endhint %}

### Etiqueta de gôndola

Esse tipo de etiqueta normalmente é utilizada para ser colocada na prateleira onde o produto se encontra.

As informações impressas nesse tipo de etiqueta são:

* Descrição, código de barras ou código interno e preço do produto.

{% hint style="info" %}
É possível alterar a descrição UN que será impressa na etiqueta. Pode ser alterada para CX, por exemplo.
{% endhint %}

![](https://981618378-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FaHl55K4HG6CuWy9zUjR9%2Fuploads%2F8c1qw0ZXR9FqzkWc9K1o%2FEtq%20gondola%20c%C3%B3digo%20de%20barras.jpg?alt=media\&token=92a7e8d0-936f-42be-a932-c97164df839d) ![](https://981618378-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FaHl55K4HG6CuWy9zUjR9%2Fuploads%2FmFYa3t28PdsekfrEJMy6%2FEtq%20gondola%20com%20codigo%20interno.jpg?alt=media\&token=b285992a-1da5-4f4a-b92b-17bf3a375049)

{% hint style="info" %}
* Ao selecionar a opção para imprimir etiqueta com código interno do produto, o código será impresso e ao final terá um digito a mais que é o digito verificador do código, como por exemplo, na primeira imagem: o código interno do produto é 123.
* Para realizar impressão desse tipo de etiqueta são utilizadas as **impressoras Zebra , Argox ou Elgin**.
{% endhint %}

A configuração da impressora para esse tipo de etiqueta é realizada através do **DMConfigurações > Impressora etiqueta**. Nesse módulo, deve estar o caminho da impressora na rede. **Exemplo: \\\terminal\Elgin**. Após inserir o caminho, selecionar o tipo de etiqueta.

![](https://981618378-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FaHl55K4HG6CuWy9zUjR9%2Fuploads%2F9jNVQVwMqdiQwwIRmsVI%2Fconfiguracao%20dm.PNG?alt=media\&token=75ea0a5b-d790-4c74-afb3-955ad3e6458a) ![](https://981618378-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FaHl55K4HG6CuWy9zUjR9%2Fuploads%2FKC1tuBwRtiUzWhBZIpP2%2FConfigura%C3%A7%C3%A3o%20impressora%20gondola.PNG?alt=media\&token=3811aeba-cdb8-46fc-aa8e-a81a4f352551)

### Etiqueta Label

Tipo de etiqueta utilizada para ser colocada em roupas.

As informações impressas nesse tipo de etiqueta são:

* Nome do estabelecimento
* Descrição, código interno e preço do produto

{% hint style="info" %}
- Nessa etiqueta ainda é impressa a seguinte mensagem: **"No caso de troca, esta etiqueta deve estar afixada ao produto. A troca deve ser realizada no prazo máximo de 15 dias.".** Esta mensagem é padrão e **não é possível alterar**.
- Para realizar impressão desse tipo de etiqueta são utilizadas as **impressoras Zebra ou Elgin**.
- Ainda não possuímos imagem desse tipo de etiqueta para inserir no manual.
{% endhint %}

A configuração da impressora para esse tipo de etiqueta é realizada através do **DMConfigurações > Impressora etiqueta**. Nesse módulo, deve estar o caminho da impressora na rede. **Exemplo: \\\terminal\Elgin**. Após inserir o caminho, selecionar o tipo de etiqueta.
