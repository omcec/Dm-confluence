# Conceitos

### DMASTER FARMA

Sistema de gestão para Drogarias, utilizado para gerenciar e controlar produtos, estoque, vendas, clientes e contas a pagar.

### DMASTER SHOP

Sistema de gestão para comércio em geral, como padarias, supermercados, depósitos, entre outros. É utilizado para gerenciar e controlar produtos, estoque, vendas, clientes e contas a pagar.

### DMASTER SNGPC

Sistema de Gerenciamento de Produtos Controlados, utilizado pelo farmacêutico para controle e gestão dos medicamentos controlados comercializados pelo estabelecimento.

### DMASTER ECF

Sistema utilizado para gerenciar e controlar o caixa do estabelecimento e responsável por realizar a impressão do documento fiscal das vendas.

### DMASTER BACKUP

Aplicativo responsável por salvar todos os dados do sistema para evitar que sejam perdidas informações caso ocorra algum problema com os equipamentos do estabelecimento.

### DMASTER NFE

Sistema que gera documento fiscal a partir de dados do DMASTER FARMA e DMASTER SHOP. A geração do documento é realizada em comunicação com o SEFAZ (Secretaria de Estado da fazenda).
