# Cote fácil

**Cote Fácil** é um empresa de cotações de produtos e o sistema realiza o processo de gerar um arquivo para que o usuário possa enviá-lo para essa empresa.

A seguir, será explicado o processo para exportação do arquivo pelo sistema DMASTER.

Para exportar toda a base de produtos do sistema DMASTER, é preciso acessar no DMASTER o módulo **Utilitários > Atualizar > Cote Fácil.**

![](https://981618378-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FaHl55K4HG6CuWy9zUjR9%2Fuploads%2FvBtQeYPBums7Xq7Xf50c%2FMenu%20cote%20facil.png?alt=media\&token=4f6b0b74-9992-466f-ab0d-4213239ffad4)

Ao acessar o módulo, será gerada a mensagem de exportação, sendo necessário clicar em **Sim.**

![](https://981618378-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FaHl55K4HG6CuWy9zUjR9%2Fuploads%2FBYlRVtPqz4JF7pTe3YGn%2Fconfirmacao.PNG?alt=media\&token=851b6f4c-8c88-4e3b-9583-2588e4886885)

Após a exportação, o arquivo será gerado na pasta **MBHSist > Cote Fácil.**

![](https://981618378-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FaHl55K4HG6CuWy9zUjR9%2Fuploads%2FHexdZGQ55TmPKxxtxJn0%2FArquivos%20exportado.png?alt=media\&token=1349f53e-ea7a-401f-a786-f1427bf55d53)

Com o arquivo exportado, o usuário deverá realizar o envio para o Cote Fácil através do site da empresa [https://cotefacil.online/](https://cotefacil.online/).

{% hint style="info" %}
A DMASTER não tem acesso e informações quanto ao processo de envio do arquivo para o Cote Fácil.
{% endhint %}

Além da possibilidade de exportar um arquivo com toda a base de dados do sistema, também é possível gerar um arquivo apenas com produtos selecionados atavés do módulo de manutenção de pedidos do DMASTER FARMA. O processo será explicado a seguir.

Acessar o módulo **Relatórios > Manutenção de pedidos.**

![](https://981618378-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FaHl55K4HG6CuWy9zUjR9%2Fuploads%2Fm3ZT5byk3jp0Td7H0X3d%2FManutencao%20de%20pedidos.png?alt=media\&token=defda785-206f-46c7-9ac3-dcf6472a1e8c)

Na tela de **manutenção de pedidos**, após selecionar o pedido desejado, escolher o **tipo de saída Cote Fácil** e o sistema irá gerar o arquivo na pasta **MBHSist > Cote Fácil.**

![](https://981618378-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FaHl55K4HG6CuWy9zUjR9%2Fuploads%2FXzh7GoiUJOupMjcYXHP5%2FManutencao%20de%20pedido%20cote%20facil.PNG?alt=media\&token=3ed94ac1-cddd-4009-851b-1c6851a0b1f2) ![](https://981618378-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FaHl55K4HG6CuWy9zUjR9%2Fuploads%2FTMOQoK7XW0YCY7ZicVfS%2FArquivo%20pedido.png?alt=media\&token=c1900291-1dbc-481d-81bc-389847059b42)
