# Integrações DMASTER

A DMASTER possui alguns tipo de integração que serão explicadas a seguir.

**Napp Solutions:** Esta empresa é responsável por inserir as informações de produtos em estoque nas pesquisas do Google. Esta integração é gratuita e nenhum processo é realizado pela DMASTER, todo o processo de integração é realizado pela empresa Napp Solutions.

**Linked Farma:** Também utilizada com a nomenclatura **IQVIA 360**, essa integração é gratuita e utilizada por estabelecimentos que fazem parte da rede universal.\
Essa integração tem o objetivo de exportar os dados de movimentações do sistema do estabelecimento para um arquivo XML e enviá-lo para a empresa LinkedFarma, para que façam os devidos procedimentos de acordo com a necessidade do cliente.\
Para mais informações sobre instalação e utilização do aplicativo, acessar [https://dmaster.gitbook.io/base-de-conhecimento-dmaster/documentacao-interna/documentacao-tecnica/broken-reference](https://dmaster.gitbook.io/base-de-conhecimento-dmaster/documentacao-interna/documentacao-tecnica/broken-reference "mention").

**AppFarma e My Farma:** Integração gratuita, com o objetivo de exportar a base de dados de produtos do estabelecimento para um arquivo .CSV, onde as informações contidas nesse arquivo serão importadas por uma empresa responsável para um tipo de loja online.\
Para mais informações sobre a instalação e utilização do aplicativo AppFarma, acessar [https://dmaster.gitbook.io/base-de-conhecimento-dmaster/documentacao-interna/documentacao-tecnica/utilitarios/dmappfarma](https://dmaster.gitbook.io/base-de-conhecimento-dmaster/documentacao-interna/documentacao-tecnica/utilitarios/dmappfarma "mention").

**Cote Fácil:** Integração gratuita que permite realizar cotação de preços de produtos, utilizando um arquivo de texto com a base de produtos da loja gerado pelo DMASTER FARMA e importando na plataforma da Cotefácil.\
Para mais informações sobre a utilização do cote fácil, acessar [https://dmaster.gitbook.io/base-de-conhecimento-dmaster/documentacao-interna/documentacao-tecnica/utilitarios/cote-facil](https://dmaster.gitbook.io/base-de-conhecimento-dmaster/documentacao-interna/documentacao-tecnica/utilitarios/cote-facil "mention").

**Recarga:** Integração gratuita que permite o estabelecimento realizar o processo de recarga de celular pelo sistema DMASTER. Para esse processo, é necessário que o estabelecimento seja credenciado com a empresa RV Tecnologia e possua os dados de integração, para posteriormente realizar a implantação.\
Essa integração também inclui a utilização de Gift Cards.

**Object Pro:** Integração gratuita com o objetivo de fidelizar clientes através de descontos em produtos fornecidos por uma Rede.\
Para mais informações sobre a instalação e utilização do aplicativo, acessar [https://dmaster.gitbook.io/base-de-conhecimento-dmaster/documentacao-interna/documentacao-tecnica/utilitarios/dm-object-pro](https://dmaster.gitbook.io/base-de-conhecimento-dmaster/documentacao-interna/documentacao-tecnica/utilitarios/dm-object-pro "mention").
