# Verificar IP no Windows

#### Como verificar IP do computador ?

Sabemos que IP é um número que identifica o computador na rede, sendo assim, este número deve ser válido, para que o computador tenha acesso a internet e esteja acessível na rede.\
Abaixo será mostrado o passo a passo de como verificar o IP do computador no Windows.

O primeiro passo é acessar o **painel de controle**, alterar a exibição para **ícones grandes** e clicar em **central de rede e compartilhamento**.

![](https://981618378-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FaHl55K4HG6CuWy9zUjR9%2Fuploads%2FO6qiJT6IEb2jjgexCb96%2Fcentral%20de%20rede%20e%20compartilhamento.PNG?alt=media\&token=d4f22670-1d4b-47c4-8cd1-b3d53883f20a)

Na próxima tela, clicar em **alterar as configurações do adaptador**.

![](https://981618378-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FaHl55K4HG6CuWy9zUjR9%2Fuploads%2F6edTKOwRncwsJ6bRtAqC%2FAlterar%20configuracoes%20do%20adaptador.PNG?alt=media\&token=2f9148a4-71ec-4981-9e05-a7287073c9dc)

Na tela de **conexões de rede**, utilizar o duplo click do mouse no registro da internet que o computador está conectado.

![](https://981618378-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FaHl55K4HG6CuWy9zUjR9%2Fuploads%2Fb9uFecamM5vE7BnQ2o06%2FRedes%20de%20internet.PNG?alt=media\&token=3ac8c0b4-8896-4d34-9678-40bb09237336)

Na tela de status, clicar em **detalhes** e na próxima tela, no campo **Endereço IPv4** será mostrado o **IP** daquele computador.

![](https://981618378-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FaHl55K4HG6CuWy9zUjR9%2Fuploads%2FSezDAto8zT4sALDDEKWI%2FDetalhes.PNG?alt=media\&token=57e85ec6-bb32-4f45-a83e-d19e8022c078)

![](https://981618378-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FaHl55K4HG6CuWy9zUjR9%2Fuploads%2F0VfAr3xRceZ6qBILopHi%2FIP.PNG?alt=media\&token=a8dd3fe5-8ccc-4e49-817a-d0a9a8e7fc9d)

{% hint style="info" %}
Caso o IP mostrado seja começado em 169, significa que Windows solicitou um endereço IP e não o recebeu. Quando isso ocorre, o computador ficará sem rede e possivelmente sem internet.\
Para correção do IP, é preciso reiniciar o aparelho da internet e computador para que o Windows consiga receber um IP válido.
{% endhint %}
