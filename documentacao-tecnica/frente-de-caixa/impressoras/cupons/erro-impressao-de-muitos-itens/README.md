# Erro impressão de muitos itens

Quando há muitos itens em uma mesma venda, pode ocorrer o erro do cupom ser cortado na posição errada antes do fim dos itens. Para corrigir o problema, é preciso:

* Criar uma nova configuração de papel com as medidas de 72.1 mm de largura e 3276 mm;
* Nomear com a descrição da marca e/ou modelo da impressora para mantermos um padrão, e;
* Selecionar esse novo parâmetro de papel que criamos como padrão.

Essas opções podem ser encontradas nas **Propriedades da impressora** do **Windows**.
