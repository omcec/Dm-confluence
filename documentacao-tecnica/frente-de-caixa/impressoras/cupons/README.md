# Cupons

No sistema de frente de caixa são realizadas impressão dos seguintes tipos de cupons:

**Cupom fiscal:** cupom principal da venda, onde contem as principais informações, como: informações do estabelecimento e do produto, forma de pagamento, valor do produto, valor de desconto, valor aproximado de impostos, número do cupom fiscal, data e hora de emissão.

**Cupom vinculado:** cupom impresso em venda de convênio, PBM e entrega. São cupons não fiscais e contem as informações do cliente, do produto e do pagamento.

**Cupom fiscal nominal:** cupom que contém os dados do cliente impressos.

Além dos cupons, também é possível imprimir comprovante de recebimento de convênio, comprovante de crédito de devolução de venda, recibo de sangria e suprimento, relatório de fechamento de caixa, entre outros.

Abaixo serão mostradas imagens das impressões que são realizadas pelo DMASTER ECF.

{% hint style="info" %}
Os cupons que são impressos no NFCe e na impressora fiscal são basicamente iguais, serão mostradas imagens dos dois casos, apenas quando houver informações exclusivas e/ou divergentes de um modelo para outro.
{% endhint %}
