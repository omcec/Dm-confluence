# PBM's

**PBM (Programa de Benefício em Medicamentos)** são acordos entre indústria, distribuidores e farmácias, oferecendo descontos especiais em diversos medicamentos para o consumidor final. Atualmente administra as compras de medicamentos de aproximadamente 200 milhões de beneficiários.

O sistema **DMASTER FARMA** suporta alguns tipos de **PBM**, são eles:

* **Big Convênios;**
* **E-Pharma;**
* **Farmácia Popular;**
* **Funcional Card;**
* **Orizon;**
* **Pharma Link;**
* **Transaction Centre (TRN Centre ou Portal da Drogaria);**
* **Vida Link.**

Para funcionarem corretamente, estes deverão ser instalados e configurados no computador do cliente.

Etapas para habilitação de cada PBM:

**1. Credenciamento;**

**2. Instalação;**

**3. Configuração;**

**4. Venda Teste;**

**5. Produção.**
