# Checklist DMASTER FARMA

Após a instalação dos sistemas DMASTER no Servidor, verificar a necessidade de instalação/configuração de aplicativos/utilitários, de acordo com a necessidade do cliente.

* [ ] Configuração de Farmácia Popular
* [ ] Instalação de PBM's
* [ ] Configuração de impressora de etiquetas
* [ ] Configuração de digitalizador de documentos
* [ ] Instalação de NF-e
* [ ] Configuração de integração entre lojas

{% hint style="warning" %}
Assuntos que são de responsabilidade de outro setor, deve ser enviado um chamado para o setor realizar a tratativa.
{% endhint %}
