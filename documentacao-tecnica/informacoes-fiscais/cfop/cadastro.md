# Cadastro

Para realizar o cadastro de um CFOP, é necessário acessar o módulo **Cadastros > Nat. Operação e Partilha ICMS > Natureza Operação**.

<figure><img src="https://981618378-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FaHl55K4HG6CuWy9zUjR9%2Fuploads%2FEy9EPWp0ZnueTLEaYBDX%2FCadastro.png?alt=media&#x26;token=eed141fa-6f68-4f76-b19b-e32c713c56d3" alt=""><figcaption></figcaption></figure>

Será gerada a tela de cadastro do CFOP e sua conversão.

<figure><img src="https://981618378-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FaHl55K4HG6CuWy9zUjR9%2Fuploads%2FxOjUM3LbgAxQMLrI0Raa%2FCadastro1.png?alt=media&#x26;token=d884b408-1c87-4d98-8bcc-5d2efb3271f2" alt=""><figcaption></figcaption></figure>

**Cfop Saída:** campo para informar o CFOP de saída;

**Cfop Entrada:** campo para informar o CFOP de entrada, de acordo com a tabela de conversão;

**Cfop Dev. Forn:** campo para informar o CFOP de devolução para fornecedor (utilizado apenas quando é necessário realizar devolução de produto de nota fiscal);

**Cfop Dev. Cliente:** campo para informar o CFOP de devolução de venda (utilizado apenas quando é necessário gerar NFe de devolução de venda);

**Descrição:** campo para informar a descrição do CFOP de acordo com a tabela de conversão;

**Finalidade de emissão:** campo para selecionar qual a finalidade do CFOP (normal, complementar, de ajuste ou devolução/retorno);

**Tipo de Operação:** campo para informar qual o tipo de operação do CFOP (operação interna, interestadual ou com o exterior).

{% hint style="info" %}
**Operação Interna:** se o fornecedor for de dentro do mesmo estado do estabelecimento;\
**Operação Interestadual:** se fornecedor for de outro estado; **Operação com Exterior:** se fornecedor for de outro país.
{% endhint %}

Após inserir todas as informações corretamente e clicar em gravar.

{% hint style="warning" %}
Uma nota pode possuir mais de um CFOP não cadastrado
{% endhint %}
