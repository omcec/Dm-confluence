# CFOP

Código Fiscal de Operações e de prestações das entradas de mercadorias e bens e da aquisição de serviços é um código do sistema tributarista brasileiro, determinado pelo governo. É indicado nas emissões de notas fiscais, declarações, guias e escrituração de livros. É utilizado em uma operação fiscal e define se a nota emitida recolhe ou não impostos, movimento de estoque e financeiro.
