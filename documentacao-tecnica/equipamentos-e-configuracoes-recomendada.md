# Equipamentos e configurações recomendada

{% hint style="danger" %}
Para qualquer procedimento de instalação do sistema, o computador **deve** possuir **Windows 10 Pro 64 bits ou versão superior** e **no mínimo 4GB de memória RAM**.
{% endhint %}

#### Configuração Servidor

<table data-full-width="false"><thead><tr><th width="227" align="center">Equipamento</th><th align="center">Configuração</th></tr></thead><tbody><tr><td align="center">Processador</td><td align="center">Core i5 ou superior (5ª geração ou superior)</td></tr><tr><td align="center">Memória RAM</td><td align="center">8GB ou superior</td></tr><tr><td align="center">Armazenamento</td><td align="center">480GB ou superior (SSD)</td></tr><tr><td align="center">Sistema Operacional</td><td align="center">Windows 10 professional ou superior</td></tr></tbody></table>

{% hint style="warning" %}
Para uso do Windows Server, necessário liberação do responsável pela rede interna da loja.
{% endhint %}

#### Configuração Terminal

<table><thead><tr><th width="225" align="center">Equipamento</th><th align="center">Configuração</th></tr></thead><tbody><tr><td align="center">Processador</td><td align="center">Core i3 ou superior (5ª geração ou superior)</td></tr><tr><td align="center">Memória RAM</td><td align="center">4GB ou superior</td></tr><tr><td align="center">Armazenamento</td><td align="center">250GB ou superior (SSD)</td></tr><tr><td align="center">Sistema Operacional</td><td align="center">Windows 10 professional ou superior</td></tr></tbody></table>

{% hint style="warning" %}
Não é recomendada a instalação do sistema em notebook. Necessária abertura de chamado pelo cliente informando que está ciente da informação.
{% endhint %}

{% hint style="danger" %}
Qualquer configuração inferior à recomendada, orientar cliente a abrir chamado que está ciente de configurações inadequadas.
{% endhint %}
