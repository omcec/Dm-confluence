# Troca de Regime Tributário

O Regime Tributário do estabelecimento é definido de acordo com seu faturamento anual. Sendo assim, pode ocorrer alteração no regime tributário do mesmo.

Para estabelecimento que realizam a troca de regime tributário, é necessário que esta informação seja alterada tanto no controle interno Dmaster quanto no sistema, pois implica em alterações de informações fiscais no cadastro de produtos.

Para realizar a alteração, devem ser seguidos os passos:

1. Encaminhar um chamado com as informações de alteração para o setor comercial, com o título **"Alteração de Dados Cadastrais"**
2. Após o chamado retornar com a alteração finalizada, entrar em contato com cliente e orientar o cliente a finalizar todas as pré vendas pendentes
3. Acessar o módulo **Utilitários > Empresa > Atualizar Dados,** para que a informação seja atualizada no sistema
4. Acessar o sistema DMASTER FARMA/SHOP no **Terminal Administrador** do estabelecimento e acessar o módulo de configurações gerais do sistema: **Utilitários > Shift + F12**
5. Após acessar o módulo, utilizar a **tecla F1,** e então será aberta a tela de atualização
6. Clicar no botão **Realizar alteração dos produtos** e aguardar até que o processo seja concluído.

![](https://981618378-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FaHl55K4HG6CuWy9zUjR9%2Fuploads%2F1QYN67j5otaOSlSD6HMw%2FAlteracao%20regime%20tributario.PNG?alt=media\&token=1874b356-b496-4f5d-a2cb-7f39bf301dc1)

Ao realizar a atualização, o sistema irá ajustar o campo **CST/CSOSN** de acordo com o ICMS informado e de acordo com o regime tributário.

7. Fechar **TODOS** os sistema em todos os computadores e abrir novamente
8. Orientar cliente a realizar venda e verificar se o cupom será impresso normalmente.

{% hint style="info" %}
Para lojas que possuem integração de cadastros, a atualização deve ser realizada apenas na MATRIZ.
{% endhint %}
