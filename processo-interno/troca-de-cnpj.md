# Troca de CNPJ

A troca de CNPJ é feita em duas etapas. Primeiro, o cliente deve acessar a 'Área do cliente' no site da DMASTER e clicar em 'Cadastre-se' para registrar o novo CNPJ e outras informações. Depois do cadastro, a equipe responsável fará o primeiro contato para orientar sobre o processo, que será finalizado pela equipe de suporte.

<summary>Processos realizados pelo setor diversos</summary>

* Receber o chamado do setor comercial referente a troca de CNPJ
* Verificar os dados de implantação
* Fechar os sistemas Dmaster em TODOS os computadores do estabelecimento
* Fazer uma cópia do banco de dados antigo e salvar em uma pasta separada no computador do cliente
* Desanexar o banco de dados antigo
* Anexar o novo banco de dados
* Configurar os sistemas em todos os computadores (Dmaster Farma/Shop, Dmaster ECF, Dmaster NFCe)
* Realizar venda de teste
* Após a finalização dos processos anteriores, seguir processos de treinamento e pós venda de acordo com [https://dmaster.gitbook.io/base-de-conhecimento-dmaster/documentacao-interna/processos-internos/instalacao-inicial](https://dmaster.gitbook.io/base-de-conhecimento-dmaster/documentacao-interna/processos-internos/instalacao-inicial "mention")
