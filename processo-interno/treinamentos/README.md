# Treinamentos

### FARMA/SHOP

O sistema FARMA/SHOP possui um cronograma de treinamento dividido em três etapas (treinamento **inicial + complementar fase 1 + complementar fase 2**, dividido em três dias ou apenas o treinamento geral em apenas um dia) onde o estabelecimento poderá escolher uma delas no ato da contratação.

#### Inicial

O treinamento inicial é normalmente o primeiro contato do estabelecimento com o sistema, é neste treinamento que é compartilhado todo o conceito básico do sistema (padrões, atalhos e etc.) e todos os módulos cadastrais que irão dar base para realizar as movimentações operacionais.

**Cronograma do conteúdo a ser ministrado disponível em:** [https://dmaster.gitbook.io/base-de-conhecimento-dmaster/documentacao-interna/processos-internos/treinamentos/conteudos](https://dmaster.gitbook.io/base-de-conhecimento-dmaster/documentacao-interna/processos-internos/treinamentos/conteudos "mention")

#### Complementar Fase 1

**Operacional**

O treinamento complementar é onde se da início a utilização de toda base cadastral realizada no último treinamento nos módulos de movimentação (pré-vendas, entradas, transferências e etc.), por isso é importante validar com o usuário antes de iniciar o treinamento se foi realizado o cadastro dos principais produtos para podermos colocar em prática neste treinamento.

**Relatórios**

O treinamento de relatórios é onde conseguimos mostrar ao usuário os resultados de todas as movimentações geradas por ele no treinamento cadastral e operacional. Estes resultados podem ser vistos através dos relatórios de grade (grid), impressos e gráficos.

**Utilitários**

O treinamento de utilitários é onde estão algumas funcionalidades do sistema que vão auxiliar o usuário no dia a dia ou que fazem certos processos pontuais. Estas funcionalidades facilitam a gestão dos produtos, gestão do estoque e demais outras funcionalidades.

**Cronograma do conteúdo a ser ministrado disponível em:** [https://dmaster.gitbook.io/base-de-conhecimento-dmaster/documentacao-interna/processos-internos/treinamentos/conteudos](https://dmaster.gitbook.io/base-de-conhecimento-dmaster/documentacao-interna/processos-internos/treinamentos/conteudos "mention")

### Complementar fase 2

O treinamento complementar fase 2, é onde é realizado toda a movimentação de finalização de venda do sistema, seja a partir de pré-vendas, vendas diretas, entregas ou integrações (PBM's).

Cronograma do conteúdo a ser ministrado disponível em: [https://dmaster.gitbook.io/base-de-conhecimento-dmaster/documentacao-interna/processos-internos/treinamentos/conteudos](https://dmaster.gitbook.io/base-de-conhecimento-dmaster/documentacao-interna/processos-internos/treinamentos/conteudos "mention")

#### Geral

O treinamento geral é ministrado normalmente para estabelecimentos que irão utilizar o banco de dados convertidos de outro sistema, ou seja, já possuem toda a base cadastral, os que já conhecem o sistema superficialmente e precisam apenas se aprofundar em módulos principais do dia a dia da do estabelecimento ou para novos funcionários que precisam aprender a utilizar o sistema.

### NFCE

O treinamento do módulo fiscal de NFCE é ministrado quando o estabelecimento está migrando para esse novo módulo de documento fiscal ou juntamente com o treinamento operacional/geral, que é normalmente o momento que os usuários estão conhecendo os módulos operacionais do sistema.

### SNGPC

O treinamento do SNGPC é ministrado normalmente para responsáveis técnicos das drogarias, onde o mesmo vai conhecer todo o processo de controle de medicamentos conforme as regras definidas pela ANVISA para este programa.

**Cronograma do conteúdo a ser ministrado disponível em:** [https://dmaster.gitbook.io/base-de-conhecimento-dmaster/documentacao-interna/processos-internos/treinamentos/conteudos/sngpc](https://dmaster.gitbook.io/base-de-conhecimento-dmaster/documentacao-interna/processos-internos/treinamentos/conteudos/sngpc "mention")

### NFE

O treinamento do NFE (Nota Fiscal Eletrônica) é ministrado no momento da instalação do módulo ou diante solicitação do estabelecimento. Neste treinamento é compartilhado todo o conceito do documento fiscal, suas particularidades e funcionalidades para emissão destes documentos.
