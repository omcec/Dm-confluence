# Site DMASTER

Para acessar o site DMASTER, é utilizado o endereço [www.dmaster.com.br](https://dmaster.com.br/).\
O site é utilizado para que os clientes realize alguns processos, como visualizar boletos, visualizar senha mensal, abrir chamados, entre outros.\
Também é utilizado pela equipe da DMASTER para realizar o download de arquivos.\
A seguir serão explicadas todas as funcionalidades do site.

Ao acessar o site e clicar em **área do cliente**, é possível visualizar algumas opções que serão explicadas a seguir.

#### Cadastre-se

Esta opção é utilizada normalmente pela equipe comercial para que o responsável por um estabelecimento cadastre os dados da loja, caso tenha interesse em se tornar um cliente DMASTER.

Esta funcionalidade também pode ser utilizada pelos clientes DMASTER quando vão trocar de CNPJ, ou realizar qualquer operação que seja necessário cadastrar um novo estabelecimento como cliente DMASTER.

Todas as orientações sobre o cadastro são passadas pela equipe comercial DMASTER e eles que dão total apoio quando a sua utilização.

Utilizando esta opção, o cliente deve preencher todas as informações corretamente e após clicar em cadastrar, as informações são enviadas para o controle interno DMASTER, ou seja, esse novo estabelecimento é cadastrado no controle.

![](https://981618378-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FaHl55K4HG6CuWy9zUjR9%2Fuploads%2FLyld5Tf77KLFkuW0TLLv%2Fcadastre-se.png?alt=media\&token=08784d51-5890-4dbc-9400-f3ebf5791947)

#### Abrir chamado

Opção utilizada para que o cliente abra um chamado quando precisar de atendimento. Ele precisará preencher os campos necessários com as informações corretas, clicar em enviar e então será criado um chamado com aquelas informações no controle para que a equipe do suporte DMASTER entre em contato com o cliente.

{% hint style="info" %}
Esta opção também pode ser acessada realizando login na área do cliente.
{% endhint %}

#### Downloads

Um processo que pode ser utilizado durante os atendimentos é baixar alguns arquivos disponíveis, caso o envio do servidor DMASTER para o computador do cliente seja muito demorado.

Para visualizar os arquivos disponíveis e posteriormente baixá-los, é preciso clicar em downloads na tela mostrada acima.

* **Instalação servidor:** executável do instalador dos sistemas DMASTER para servidor;
* **Instalação terminal:** executável do instalador dos sistemas DMASTER para terminal;
* **Suporte DMASTER:** executável do arquivo do Team Viewer da DMASTER, onde não é necessário instala-lo;
* **Team Viewer:** executável de instalação do arquivo para conexão remota, o Team Viewer;
* **Microsoft.NET Framework 4:** executável do arquivo de instalação do Net Framework 4, que é necessário para instalação dos instaladores dos sistemas DMASTER;
* **Integração Farmácia Digital:** executável do instalador ao aplicativo de integração da Farmácia Digital.

Para baixar algum arquivos, é preciso clicar na palavra **download** correspondente ao executável desejado e o download será iniciado.

{% hint style="info" %}
Esta opção também pode ser acessada realizando login na área do cliente.
{% endhint %}

#### Boletos

Opção utilizada para que o cliente visualize os boletos referente aos serviços/sistemas DMASTER para o seu estabelecimento. Caso o boleto ainda não tenha sido pago, é possível imprimir/salvar para realizar o pagamento. Para verificar os boletos, é preciso, digitar o CNPJ do estabelecimento e clicar em ok.

{% hint style="info" %}
Esta opção também pode ser acessada realizando login na área do cliente.
{% endhint %}

Ao realizar o login é exibido na tela inicial, o nome do estabelecimento juntamente com sua senha mensal. E logo abaixo, as formas de contato da DMASTER.\
Ao lado, são mostradas ao opções que podem ser utilizadas pelo cliente. Estas serão explicadas a seguir.

#### Contratos

Opção utilizada para que o cliente confirme o contrato de instalação dos sistemas DMASTER. Ao clicar na opção, serão exibidas as informações do contrato e ao clicar em visualizar, o contrato será aberto.\
Para confirmá-lo, é preciso clicar no botão eu concordo com os termos.

#### Sugestões

Opção utilizada para que o cliente dê sugestões de melhorias para os sistemas DMASTER.\
Para criar uma sugestão, é preciso clicar no botão nova sugestão, preencher as informações necessárias e clicar em gravar. Após esse processo, a sugestão será registrada e a equipe avaliará se a sugestão é viável para ser implementada no sistema.
