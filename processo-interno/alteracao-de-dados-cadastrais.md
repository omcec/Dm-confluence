# Alteração de dados cadastrais

Para estabelecimento que realizam alteração de dados cadastrais (telefone, endereço e/ou e-mail), é necessário que esta informação seja alterada tanto no controle interno Dmaster quanto no sistema.

Para realizar a alteração, devem ser seguidos os passos:

1. Encaminhar um chamado com as informações de alteração para o setor comercial, com o título **"Alteração de Dados Cadastrais";**
2. Após o chamado retornar com a alteração finalizada, entrar em contato com cliente e conectar no **Terminal Administrador**;
3. Acessar o módulo **Utilitários > Empresa > Atualizar Dados,** para que a informação seja atualizada no sistema;
4. Após a alteração, todos os sistemas Dmaster devem ser reiniciados em todos os computadores do estabelecimento.
