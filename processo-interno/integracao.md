# Integração

### Configuração de porta no modem/roteador

Um dos principais requisitos para configuração de acesso online/integração de produtos é que a porta 1666 esteja aberta nas configurações do modem/roteador da loja.

#### Cliente abre chamado solicitando configuração de acesso online/integração de produtos

Caso o cliente entre em contato solicitando uma das funcionalidades citadas acima, deve-se verificar primeiro se a porta encontra-se aberta. Caso o resultado seja negativo, ou seja, a porta esteja fechada, o processo a ser realizado é:

1. Informar ao cliente que solicite a abertura da porta para o provedor da internet;
2. Orientar que ele abra um novo chamado quando o processo de abertura da porta for concluído;
3. Finalizar o chamado.

### Cliente abre chamado solicitando configuração de integração de produtos

{% hint style="warning" %}
Na configuração desse tipo de integração, caso a loja possua movimentações no sistema, ao realizar essa configuração, elas serão perdidas. Com isso, antes de iniciar o processo de configuração, devem ser realizados os processos que são explicados abaixo.
{% endhint %}

O principal requisito para que seja realizada a configuração de integração de produtos é que o acesso online já esteja configurado e funcionando corretamente. Caso o requisito não seja atendido, deve-se realizar os processos de configuração de acordo com [https://dmaster.gitbook.io/base-de-conhecimento-dmaster/documentacao-interna/documentacao-tecnica/integracao-entre-lojas/acesso-online](https://dmaster.gitbook.io/base-de-conhecimento-dmaster/documentacao-interna/documentacao-tecnica/integracao-entre-lojas/acesso-online "mention") e validar se está funcionando.

Após a validação do funcionamento do acesso online, o chamado deve ser encaminhado para o responsável (Jairo), para que realize as devidas tratativas.\
Com as validações concluídas, o responsável irá enviar o chamado para o setor novamente, com as informações atualizadas. Neste momento, o processo de configuração já pode ser realizado de acordo com [https://dmaster.gitbook.io/base-de-conhecimento-dmaster/documentacao-interna/documentacao-tecnica/integracao-entre-lojas/integracao-de-produtos](https://dmaster.gitbook.io/base-de-conhecimento-dmaster/documentacao-interna/documentacao-tecnica/integracao-entre-lojas/integracao-de-produtos "mention").
