# Backup Local

O backup local deve ser usado apenas quando o cliente não tem outra máquina disponível para armazenar o backup. É fundamental instruir o cliente a fazer cópias externas desses arquivos, pois, se a máquina principal apresentar algum problema que impeça o acesso aos dados, essas cópias serão essenciais para recuperar o sistema.

#### Configurando o Backup Local

Para configurar o backup local:

1. Acesse o **DMConfigurações** e clique no ícone de backup.
2. No campo "**Caminho do backup**", insira o endereço local para salvar o backup (exemplo: `"C:\MCBKP"`).
3. Configure as opções de "**Atualiza IP**" e "**Porta SQL**" de acordo com a rede de integração do estabelecimento.
